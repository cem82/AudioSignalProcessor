# AudioSignalProcessor

An interactive audio processing application developed as an EE101 Term Project.

Signal Manipulation: Modify audio signals through speed adjustment, amplitude scaling, and vector flipping (reverse playback).

Recording & Visualization: Capture multiple sounds, visualize them on a real time graph, and perform comparisons.

Storage: Export and save processed audio files in .wav format.
